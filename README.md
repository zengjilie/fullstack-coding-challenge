# OH MY BIBLE

Demo Site: https://ohmybible.herokuapp.com/

## Tech Stack
Front End: JavaScript, HTML/CSS, SASS

Back End: NodeJS, ExpressJS, EJS

Testing: Jest

npm dev using nodemon

npm test
